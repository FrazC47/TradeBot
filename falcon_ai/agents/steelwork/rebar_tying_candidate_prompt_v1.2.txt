WHAT COUNTS AS ONE COMPLETE TIE (CRITICAL DEFINITION)
================================================================================

A "New Tie Event" is ONE complete rebar tying action with these MANDATORY components:

START: Wire makes first contact with rebar intersection
END: Final twist completed AND pigtail cut/tucked

ONE TIE = ONE continuous sequence including:
- Wire positioning (if done immediately before contact)
- Wrapping/twisting motion
- Final securing
- Pigtail finishing

================================================================================
WHAT IS NOT A TIE (DO NOT COUNT THESE)
================================================================================

| Action | Why NOT a Tie |
|--------|---------------|
| Re-gripping pliers | Same tie in progress, tool adjustment only |
| Pausing to reposition | Same tie in progress, brief pause |
| Checking/pulling a tie | Inspection AFTER tie is complete |
| Moving to next bar | Material handling between ties |
| Getting more wire | Material preparation, not tying |
| Adjusting body position | Still working on same tie |
| Re-attempting same tie | Same intersection, correcting error |
| Partial/incomplete wrap | Abandoned and restarted = same tie location |

VISUAL RULE: A new tie ONLY starts when wire contacts a NEW intersection.
If wire is at the SAME intersection, it is the SAME tie (even if paused/adjusted).
