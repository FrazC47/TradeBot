# 2026-02-28 - Strategy Optimization Session

## Trading Strategy Development

### Baseline Strategy (Working)
- **Timeframe alignment**: All 3 timeframes must agree (1h, 4h, 1d)
- **Daily trend filter**: Only trade in direction of daily EMA trend
- **Fibonacci entries**: 0.618 and 0.5 for optimal reversal zones
- **RSI filter**: Avoid extremes (>70 for longs, <30 for shorts)
- **Volume filter**: Minimum 80% of 20-period average
- **R:R ratio**: 1:2 minimum (1.5x stop, 3x target)
- **Holding period**: 20 candles (20 hours for 1h entries)
