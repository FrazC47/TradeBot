# 🎯 Kimi Claw Headhunting

Welcome to the headhunting workspace for talent acquisition and recruitment research.

## 📁 Project Structure

```
headhunting/
├── README.md           # This file
├── leads/              # Candidate leads and prospects
│   ├── active/         # Currently engaged candidates
│   └── cold/           # Prospects not yet contacted
├── outreach/           # Outreach messages and templates
│   ├── sent/           # Sent messages (archived)
│   └── drafts/         # Draft messages in progress
├── research/           # Market and company research
│   ├── companies/      # Company profiles and org charts
│   ├── roles/          # Role research and JD analysis
│   └── markets/        # Industry/talent market insights
├── pipelines/          # Active hiring pipelines
│   └── YYYY-MM-DD-role.md
└── templates/          # Reusable templates
    ├── email-templates.md
    ├── linkedin-templates.md
    └── tracking-sheets.md
```

## 🚀 Quick Start

1. **Start a search**: Create a new research file in `research/roles/`
2. **Track candidates**: Add leads to `leads/active/` or `leads/cold/`
3. **Draft outreach**: Use templates in `templates/` for consistent messaging
4. **Manage pipeline**: Create pipeline files for active searches

## 📝 Naming Conventions

- **Research files**: `YYYY-MM-DD-company-or-role.md`
- **Lead files**: `firstname-lastname-role.md`
- **Pipeline files**: `YYYY-MM-DD-role-company.md`
- **Outreach files**: `YYYY-MM-DD-candidate-name.md`
